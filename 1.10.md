### Q 1.10

```
gosh> (define (A x y) (cond ((= y 0) 0) ((= x 0) (* 2 y)) ((= y 1) 2) (else (A (- x 1) (A x (- y 1))))))
A
gosh> (A 1 10)
1024
gosh> (A 2 4)
65536
gosh> (A 3 3)
65536
```

`n < 2`のとき
```
A(k,0) = 0
A(k,1) = 2
```

`n >= 2`のとき
```
A(0,n) = 2n
A(1,n) = A(0, A(1, n-1)) = 2A(1, n-1) = ... = 2^n
A(2,n) = A(1, A(2, n-1)) = 2^(A(2, n-1)) = 2^(2^(2^(....2))) [n個の指数] = 2↑n (↑: クヌースの矢印)
A(3,n) = A(2, A(3, n-1)) = 2↑(A(3, n-1)) = 2↑(2↑(A(3, n-2))) = .... = 2↑↑n
A(k,n) = 2 ↑^k n 
```

よって一般に`A(k,n) = 2 ↑^k n`

↑ はクヌースの矢印記号
