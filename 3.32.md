```
(define (and-gate a b c))
```

(a の変更, bの変更) の順に次第書きを登録したとして

* queue の場合

```
(a, b, c)
-> (0, 1, 0) ; 0 step
-> (1, 1, 1) ; a の変更 + propagate
-> (0, 1, 0) ; b の変更 + propagate
```

* (FILO) の場合

```
(a, b, c)
-> (0, 1, 0) ; 0 step
-> (0, 0, 0) ; b の変更 + propagate
-> (1, 0, 0) ; a の変更 + propagate
```

c の振る舞いが異なる！！
